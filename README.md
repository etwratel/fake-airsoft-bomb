# fake-airsoft-bomb
## Source code for a fake bomb running on Arduino.

Author : etwratel  
Version : 1.2  
Release Date : 23/07/2019  

## Requirements :
- Arduino Uno (for nano or PJRC Teensy, please ajust the pin mapping)
- 2 pushbuttons
- 1 LCD display 16x2
- 1 switch
- 1 buzzer
- 3 resistors 330Ohms
- 1 capacitor 100uF
- 1 battery 9V
(Most of its components are in the Arduino nano starter kit.)

## How to use it ?
1) Download/clone the source code
2) Connect your Arduino to your computer
3) Run Arduino IDE and open the downloaded code
4 ) Select the following settings for compilation :
-- Board type : Arduino Uno (or nano or Teensy 3.1/3.2)  
-- USB Type : Keyboard  
-- CPU Speed : 96Mhz or 120Mhz  
-- Port : /dev/hidraw  
5) Upload the code on the board
6) Connect the different elements according to the provided diagram

Enjoy !
